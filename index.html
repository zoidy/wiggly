Welcome to Wiggly’s documentation!¶

Wiggly implements an uncertainty representation and analysis framework for models with fuzzy-probabilistic uncertainty, focusing on GIS applications with uncertain points, lines or polygons. The source code is available at [https://github.com/zoidy/wiggly].

This software is distributed under the LGPL v3.
