gis.wiggly_qgisÂ¶

Adapter which allows using QGIS scripts (including Processing scripts) as analysis models in Wiggly.

Requirements:
QGIS 2.4 or above A working Wiggly installation in the QGIS-bundled Python
Usage:
See the example in the examples directory.
Notes:

Don’t name your Processing scripts ‘qgis.py’ or ‘test.py’ or anything too common as those kinds of names may interfere with QGIS and cause Wiggly not to work as expected.

Your processing scripts must contain a function which has the operations you wish to analyze. In order for your processing script to be runnable by processing as well as this library, it is recommended to place all script code in a function called go() which takes in as parameters any required inputs. See the qgis example for more details
